**Coffee Sales Dashboard
**
This project contains a Power BI dashboard created to analyze coffee sales performance across different time periods and categories.
All data was cleaned and prepared using Power Query before building the visuals.

Files Included: 
Coffee Sales.pbix – Final interactive dashboard,
Data_set.xlsx – Raw data used for the dashboard

Project Summary: 
The dashboard provides a clear view of total coffee sales and helps understand how sales vary by hour, weekday, coffee type, and time of day.
It includes slicers and visuals that allow users to explore the data easily.

Features: 
Sales trend over time,
Hour-of-day filter,
Weekday sales comparison,
Coffee type performance,
Payment method split,
Time-of-day analysis,
Clean and interactive layout,

Data Cleaning (Power Query):
Removed duplicates,
Handled missing values,
Standardized column names,
Transformed date and time fields,
Created new columns (weekday, time of day),
Filtered unnecessary rows

Tools Used: 
Power BI,
Power Query,
DAX (basic measures).
