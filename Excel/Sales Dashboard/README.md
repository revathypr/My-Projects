**Sales Dashboard**
This project contains an Excel dashboard created to analyze overall sales performance across different categories and time periods.

Files Included: 
Sales_Dashboard.xlsx – Final interactive Excel dashboard
Sales_Data.xlsx – Data used for analysis

Project Summary: 
The dashboard provides a clear view of sales performance and helps identify trends, category contribution, and key metrics.
It includes interactive elements to quickly understand how sales change based on different filters.

Features: 
Sales trend charts
Category-wise performance visuals
KPI indicators (Total Sales, Orders, Revenue, etc.)
Slicers for easy filtering
Clean layout for quick insights

Tools Used: 
Microsoft Excel
Pivot Tables & Pivot Charts
Data Cleaning
Slicers & Formatting
