**Cookie Performance Dashboard**

This project contains an Excel interactive dashboard built to analyze the performance of different cookie categories.

Files Included
Cookies Performance Dashboard.xlsx – Final interactive dashboard
Data Set.xlsx – Raw data used for analysis

Project Summary
The dashboard provides a simple view of cookie sales performance.
It includes charts, slicers, and KPIs to understand which cookie categories perform best and how sales vary across segments.

Features
Category-wise performance charts
Dynamic slicers for filtering
KPIs such as total sales and orders
Clean and easy-to-use layout

Tools Used
Microsoft Excel
Pivot Tables & Pivot Charts
Slicers
Basic formulas
